Alright Shrey, I can see your project structure already has the right files laid out. Here's the plan — clean, minimal,no over-engineering. You need four pieces:

**1. `voice/stt.py`** — Captures mic audio and runs Whisper on it:

```python
import whisper
import sounddevice as sd
import numpy as np
import tempfile
import scipy.io.wavfile as wav

model = whisper.load_model("base")

def listen(duration=5, sample_rate=16000):
    print("Listening...")
    audio = sd.rec(int(duration * sample_rate), samplerate=sample_rate, channels=1, dtype="float32")
    sd.wait()
    audio = np.squeeze(audio)
    result = model.transcribe(audio, fp16=False)
    return result["text"].strip()
```

**2. `voice/tts.py`** — Speaks the response out loud:

```python
import pyttsx3

engine = pyttsx3.init()
engine.setProperty("rate", 175)

def speak(text):
    engine.say(text)
    engine.runAndWait()
```

**3. `agent.py`** — Sends text to Claude and gets a response (direct API, no LangChain per your preferences):

```python
import anthropic

client = anthropic.Anthropic()

def ask(prompt, system_prompt="You are Alfred, a concise personal AI assistant."):
    response = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=1024,
        system=system_prompt,
        messages=[{"role": "user", "content": prompt}]
    )
    return response.content[0].text
```

**4. `main.py`** — The voice loop that ties it all together:

```python
from voice.stt import listen
from voice.tts import speak
from agent import ask

def main():
    speak("Alfred online. How can I help?")
    while True:
        text = listen()
        if not text or len(text) < 2:
            continue
        print(f"You: {text}")
        if "goodbye" in text.lower() or "shut down" in text.lower():
            speak("Shutting down. Goodbye Shrey.")
            break
        response = ask(text)
        print(f"Alfred: {response}")
        speak(response)

if __name__ == "__main__":
    main()
```
